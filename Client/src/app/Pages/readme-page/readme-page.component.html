Instructiuni

Pentru folosirea siteului DentMedical

Register

Pentru a creea un cont nou, completati casutele libere si apasati pe butonul de register

Login

Pentru a va loga, apasati pe butonul de login si completati cu e-mailul si parola dumneavoastra.

Felicitari! Acum faceti parte din clientela noastra selecta.

Navigatie

In partea de sus a paginii se pot observa butoanele ce ne vor plimba prin aplicatie.

Butonul Home - ne va readuce la pagina principala
    
    Butonul Appointment - este sectiunea unde va veti programa
    
    Butonul Services - unde puteti privi serviciile oferite de clinica noastra
    
    Butonul Prices - sectiunea unde va veti uimii de preturile noastre mici
    
    Butonul Contact - pentru locatia si datele de contact

Appointment

Alegeti Data si Orele libere pentru o programare, impreuna cu un mesaj despre problema Dumneavoastra.

Logout

Pentru a parasi aplicatia folositi butonul de logout!
